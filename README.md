# luadist-lustache

This project is a metapackage to install [lustache](https://github.com/Olivine-Labs/lustache) in [luadist](luadist.org)

# Install

```bash
$ git clone https://github.com/azukiapp/luadist-lustache
$ luadist install luadist-lustache
```
